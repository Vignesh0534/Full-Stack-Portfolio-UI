# Full-Stack-Portfolio-UI

# 🚀 Vignesh Dasari - Software Engineering Portfolio

A high-performance, single-page portfolio engineered to showcase full-stack development capabilities and machine learning integrations. 

## 🏗️ Architecture & Tech Stack

This project was built with a strict focus on performance, accessibility, and modern UI/UX principles, utilizing a zero-dependency architecture.

* **Frontend Structure:** HTML5
* **Styling Engine:** Tailwind CSS (Utility-First, Glassmorphism, CSS Gradients)
* **DOM Manipulation & State:** Vanilla JavaScript (ES6)
* **Theme Management:** LocalStorage API & Tailwind Dark Mode Engine
* **Deployment:** GitHub Pages (Serverless)

## ⚡ Core Features

* **Dynamic Data Rendering:** Project components are decoupled from the HTML and injected dynamically via JavaScript state arrays, mimicking a headless CMS structure.
* **Persistent Theme Toggle:** Features a custom-built Light/Dark mode transition switch that securely caches user preferences locally.
* **Responsive Grid System:** Fully fluid layout adapting seamlessly from mobile viewports to ultra-wide desktop monitors.

## 📂 Local Development Setup

To run this project locally, no heavy node modules or build steps are required.

1. Clone the repository: `git clone https://github.com/Vignesh0534/portfolio-website.git`
2. Open the directory in your code editor (e.g., VS Code).
3. Launch the `index.html` file utilizing a local development server (like the VS Code Live Server extension) to ensure the local storage API runs correctly.
